# 团队共享知识库

所有 agent 每次唤醒都会自动读到本文件（通过各自工作目录 CLAUDE.md 的 @ 引用）。
把需要全员知道的长期知识写在这里：John 的偏好、项目背景、协作约定等。
保持精炼——这里的每个字都会进入每个 agent 的上下文。

## 团队共识
- **git 提交规范**：本仓库（及 John 名下其他仓库）提交一律使用 John 的 git 身份（JRJRJPRO），commit message 不携带任何工具生成的签名或合著 trailer（如 Co-Authored-By），保持提交历史干净统一。
- **本文件是公开的**：AgentChat 已开源（github.com/JRJRJPRO/AgentChat），shared/TEAM.md 随代码一起公开。往这里写内容时不得包含 John 的隐私（联系方式、证件/签证、成绩、求职细节等）——这类信息写进各自工作目录的记忆包（不入库）。
- **日期型资料先核实再写档**：比赛/活动/招聘清单常有年份错位（把 2023/2024 的事件当成今年的）。写档前必须 WebSearch/WebFetch 核实年份、是否仍开放、参赛资格；模糊日期（"May 26"无年份）、"2.0"版本号、历史报名数都是危险信号。详细案例见记忆包「学业求职」的 feedback_verify_dated_aggregators.md
